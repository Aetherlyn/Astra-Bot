# Astra Bot

A personal Discord bot project I made to help manage D&D combat through Discord.

The goal of this project is to create a convenient way for players and Dungeon Masters to quickly access and manage character information during combat.

This is my first major project and is heavily experimental, mainly built for learning, practice, and fun.

Built using Pycord.

## Requirements

- Python 3.13+

## Installation

See the [setup guide](docs/setup.md) for step by step instructions.

## Commands

Use the `!help` command to see all available commands.

A detailed command library will be added soon for more information.
